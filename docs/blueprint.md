# **App Name**: Wallet XChanger

## Core Features:

- Payment Method Selection: Allow users to select their payment method: Bkash, Nagad, PayPal, Payoneer, or Wise.
- Payment Amount Input: Enable users to input the payment amount, with currency displayed based on the selected payment method (BDT for Bkash/Nagad, USD for others).
- Withdrawal Method Selection: Allow users to select their withdrawal method: Bkash, Nagad, PayPal, Payoneer, or Wise.
- Exchange Rate Display: Display the amount the user will receive based on the admin-defined exchange rate.
- Payment Instructions/API Integration: Show payment instructions based on the selected payment method or integrate APIs for automated payment processing.
- Payment Status Display: Display the payment status: Complete, Processing, or Paid.
- Admin Dashboard: Provide an admin dashboard to view payment information, update exchange rates, and manage user transactions.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB), providing a sense of trust and security in financial transactions.
- Background color: Light gray (#F0F8FF) for a clean, modern, and user-friendly interface.
- Accent color: Light green (#90EE90) to highlight successful transactions and important actions.
- Font: 'Inter', sans-serif, for a clean, modern, and easily readable interface.
- Use simple, consistent icons to represent different payment methods and transaction statuses.
- Use a clean and intuitive layout to guide users through the payment and withdrawal processes smoothly.
- Subtle animations to provide feedback on user interactions, such as loading animations or confirmation messages.
